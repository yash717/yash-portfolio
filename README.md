## Getting Started

### Prerequisites

- [Bun](https://bun.sh/) (v1.0+)

**Install Bun (if not installed):**

```bash
curl -fsSL https://bun.sh/install | bash
# Restart your terminal or run: source ~/.zshrc  (or ~/.bashrc)
```

### Installation

**Using Bun (recommended):**

```bash
# Clone the repository
git clone https://github.com/yash717/yash-portfolio.git
cd yash-portfolio

# Install dependencies using Bun
bun install

# Start the development server
bun dev
```

**Using npm (alternative):**

```bash
git clone https://github.com/yash717/yash-portfolio.git
cd yash-portfolio
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
# With Bun
bun run build
bun start

# With npm
npm run build
npm start
```

### Scripts

| Command | Description |
|---------|-------------|
| `bun dev` / `npm run dev` | Start development server |
| `bun run build` / `npm run build` | Build for production |
| `bun start` / `npm start` | Start production server |
